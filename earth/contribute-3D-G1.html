Protocole N²
Chaque station Astroport publie vers ses N voisins (NOSTR). Ces N voisins publient à leur tour vers N autres. N nœuds génèrent :
N × N = N²

❓ Questions techniques Ğ1
Comment le webhook OC déclenche-t-il l'émission sur Duniter ?▾
À réception d'un paiement confirmé sur OpenCollective, le script oc2uplanet.sh appelle l'API Duniter pour émettre les ẐEN sur la clé publique du contributeur. La transaction est immédiatement inscrite sur la blockchain Ğ1. Pour Cloud Usage, l'émission est instantanée via webhook. Pour les Parrains ♥BOX, elle est groupée lors du cycle mensuel.
ẐEN et ẑen sont-ils fongibles entre eux ?▾
Oui, via le Pont de Liquidité. La conversion ẐEN (propriété) ↔ ẑen (usage) est possible dans les deux sens. La fongibilité permet de détecter les chaînes de création de valeur : si vos ẐEN grossissent anormalement, c'est un signal comptable visible — la "croissance imposable" décrite dans le modèle.
Quelle relation entre NOSTR et le protocole N² ?▾
UPlanet utilise le protocole NOSTR (Notes and Other Stuff Transmitted by Relays) pour la diffusion pair-à-pair. Chaque station Astroport est un relai NOSTR. Lorsqu'un utilisateur publie un contenu, son relai local le propage à ses N relais voisins (amis), qui propagent à leurs N voisins (amis des amis). C'est la relation N² : N nœuds × N connexions = N² chemins garantis.
Le ẐEN est-il soumis à la TVA ou à l'impôt ?▾
Les jetons ẐEN/ẑen sont des tokens comptables internes, pas des instruments financiers. Le fait générateur fiscal est uniquement le remboursement en € (note de frais via OpenCollective). Statut recommandé pour les contributeurs actifs : Micro-Entrepreneur BNC. Les ẐEN non convertis ne constituent pas un revenu imposable.
