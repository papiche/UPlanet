❓ Questions fréquentes
Je n'ai pas encore de MULTIPASS, puis-je souscrire ? ▾
Oui ! Pour les tiers Satellite et Constellation, le MULTIPASS est créé automatiquement à la première contribution (PRIMO TX 1Ğ1). Pour Cloud Usage et Membre résident, créez d'abord votre MULTIPASS via le terminal UPlanet ou l'app Ẑinkgo.
Quand mes ẐEN sont-ils crédités ? ▾
Pour Cloud Usage : dès réception du paiement via webhook OC (quelques minutes). Pour les autres tiers : lors du cycle mensuel automatique de la station (script oc2uplanet.sh).
Quelle est la différence entre Parrain et Locataire ? ▾
Parrain / Co-Bâtisseur (♥BOX) : votre contribution finance une machine physique. Vos ẐEN sont émis sur une ZEN Card, puis répartis 33/33/33/1%. La machine est opérée par un Armateur et un Capitaine.

Locataire (Cloud / Membre) : votre contribution est convertie directement en ẐEN d'usage sur votre MULTIPASS. Vous les dépensez pour les services cloud.
Comment convertir ẐEN → € (remboursement) ? ▾
Via le Pont de Liquidité : transférez vos ẐEN vers uplanet.G1 avec le commentaire RESTITUTION:INDEMNISATION, puis déposez une note de frais sur OpenCollective. Le collectif valide et effectue le virement SEPA. Statut recommandé : Micro-Entrepreneur BNC.
Que finance concrètement ma contribution ♥BOX ? ▾
Votre contribution arrive sur le compte du G1FabLab#monnaie-libre. Le collectif achète la machine physique (Satellite 2To ou Hub GPU). Les ẐEN sont répartis : 33% crédits usage MULTIPASS · 33% R&D Astroport.ONE · 33% Actifs durables (forêts-jardins) · 1% Capitaine.
