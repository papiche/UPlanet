❓ QUESTIONS FRÉQUENTES

Je n'ai pas encore de MULTIPASS, puis-je souscrire ? ▾
Oui ! Pour les tiers Satellite et Constellation, le MULTIPASS est créé automatiquement à la première contribution (PRIMO TX 1Ğ1). Pour Cloud Usage et Membre résident, créez d'abord votre MULTIPASS via le terminal UPlanet ou l'app Ẑinkgo.
Quand mes ẐEN sont-ils crédités ? ▾
Pour Cloud Usage : dès réception du paiement via webhook OC (quelques minutes). Pour les autres tiers : lors du cycle mensuel automatique de la station (script oc2uplanet.sh).
Quelle est la différence entre Parrain et Usager (Locataire) ? ▾
Parrain / Co-Bâtisseur (♥BOX Satellite / ♥BOX GPU) : votre contribution finance l'achat d'une machine physique. Vos ẐEN sont émis sur une ZEN Card (jeton de propriété), puis distribués automatiquement : 33% crédits usage MULTIPASS, 33% R&D, 33% Actifs physiques, 1% Capitaine. La machine est opérée par un Armateur et un Capitaine de l'essaim.

Usager / Locataire (Cloud Usage / Membre résident) : votre contribution est convertie directement en ẐEN d'usage sur votre MULTIPASS. Vous les dépensez pour les services cloud (stockage, IA, NOSTR). Pas de lien avec l'infrastructure physique.
Comment se faire rembourser (convertir ẐEN → €) ? ▾
Via le Pont de Liquidité (service de remboursement G1FabLab) : transférez vos ẐEN vers uplanet.G1 avec le commentaire RESTITUTION:INDEMNISATION, puis déposez une note de frais justifiée sur OpenCollective. Le collectif valide et effectue le virement SEPA. Le fait générateur fiscal est le remboursement en €, pas les jetons. Statut recommandé : Micro-Entrepreneur BNC.
Que finance concrètement ma contribution ♥BOX ? ▾
Votre contribution (50€/an ou 540€/3ans) arrive sur le compte du G1FabLab#monnaie-libre.
Le collectif achète la machine physique (Satellite ou Hub GPU).
