❓ QUESTIONS FRÉQUENTES

Je n'ai pas encore de MULTIPASS, puis-je souscrire ? ▾
Oui ! Pour les tiers Satellite et Constellation, le MULTIPASS est créé automatiquement à la première contribution (PRIMO TX 1Ğ1). Pour Cloud Usage et Membre résident, créez d'abord votre MULTIPASS via le terminal UPlanet ou l'app Ẑinkgo.
Quand mes ẐEN sont-ils crédités ? ▾
Pour Cloud Usage : dès réception du paiement via webhook OC (quelques minutes). Pour les autres tiers : lors du cycle mensuel automatique de la station (script oc2uplanet.sh).
Quelle est la différence entre sociétaire et locataire ? ▾
Sociétaire (Satellite/Constellation) : vous devenez copropriétaire de la coopérative SCIC. Vos ẐEN sont dans une ZEN Card coopérative (non remboursables immédiatement). Vous avez un droit de vote aux AG.

Locataire (Cloud/Membre) : vos ẐEN sont dans un MULTIPASS d'usage. Vous les dépensez pour les services. Aucun lien coopératif.
Puis-je me faire rembourser ? ▾
Via le système de Restitution : envoyez vos ẐEN vers le portefeuille uplanet.G1 avec le commentaire RESTITUTION:INDEMNISATION, puis déposez une note de frais sur OpenCollective. Le remboursement € est validé par l'admin de la coopérative.
Que finance concrètement ma contribution ? ▾
L'infrastructure physique décentralisée : stations Raspberry Pi / serveurs home, stockage IPFS distribué, relais NOSTR, GPU pour les services IA locaux. La répartition coopérative : 1/3 Trésorerie (cloud opérations) · 1/3 R&D (développement G1FabLab) · 1/3 Actifs (forêts-jardins et biens communs).
