MineLife — Guide de démarrage

Certifiez vos savoir-faire, forgez des compétences composites et partagez vos tutoriels —
de façon décentralisée sur NOSTR, sans autorité centrale.

1
Cliquez sur « Connecter » (coin supérieur droit).
Votre MULTIPASS NOSTR s'ouvre — il contient votre clé publique et vos compétences.
2
L'onglet « Mes Compétences » se peuple.
Chaque badge affiché est un certificat signé par vos pairs sur le relay NOSTR local.
3
Un badge affiche 📹 ? → cliquez dessus pour voir la vidéo Grimoire générée lors de la certification.
Si absent, demandez #badge linux à BRO — il génère l'image via ComfyUI.
1
Onglet « Explorer » → choisissez un Permit qui vous correspond.
Un Permit décrit les ingrédients (skills requis) et le niveau minimum.
2
Cliquez « Aspirer » → un Kind 30501 est publié sur le relay.
→ BRO notifie les pairs qui détiennent déjà ce skill. Ils peuvent vous adouber.
3
Validation WoTx2 — deux voies :
Règle A 3 pairs X1+ réagissent avec ✚ → vous signez vous-même votre certificat (Kind 30503).
Règle B 1 pair vous adoube directement (Kind 30502) → certificat immédiat.
4
Le badge apparaît dans « Mes Compétences ».
→ L'Oracle de la station peut co-signer pour une attestation coopérative officielle.
1
Onglet « Atelier » → vos crafts composites disponibles s'affichent.
Un craft composite combine plusieurs skills en un nouveau Permit plus avancé.
2
Vérifiez les ingrédients — chaque skill requis doit être validé à X1+.
Les cases vertes = acquises, oranges = niveau insuffisant, rouges = absentes.
3
Tous les ingrédients réunis ? → bouton Synthétiser devient actif.
Cliquez — votre station publie le Kind 30503 de certification composite.
4
Le Grimoire se déclenche automatiquement 🎬
FFmpeg génère une vidéo badge (Ken Burns + narration TTS) publiée en Kind 22 sur NOSTR. Demandez #tts linux à BRO pour créer la narration audio avant le craft.
1
Enregistrez un tutoriel via vdo.copylaradio.com (bouton LIVE dans la topbar).
Cliquez LIVE → Démarrer → votre caméra/écran est partagé. Téléchargez la vidéo à la fin.
2
Ouvrez le Studio (bouton ✂️ Studio dans LIVE, ou « Ajouter une ressource » dans Formation).
Glissez votre fichier WebM ou MP4 dans la zone de dépôt.
3
Découpez avec les sliders Début / Fin — l'aperçu se met à jour en temps réel.
FFmpeg WASM tourne dans votre navigateur — aucune donnée ne quitte votre machine avant validation.
4
Renseignez un skill (optionnel) → cliquez ✂️ Couper & Publier.
→ Upload IPFS sur votre station. → Événement Kind 30504 publié et visible dans l'onglet Formation de tous les membres de la constellation.
1
BRO 🤖 est votre instructeur de guilde — cliquez sur le bouton violet en bas à droite.
Il connaît votre progression, votre station et la constellation NOSTR.
2
Commandes clés :
#badge linux → génère une image badge via ComfyUI
#tts linux → synthèse vocale Orpheus pour la narration Grimoire
#rec devops → recommande des formations disponibles
#mem ma note → sauvegarde une note dans votre uDRIVE
3
BRO répond sur le relay NOSTR — les réponses arrivent même si vous fermez l'onglet.
Les jobs lourds (ComfyUI, TTS, indexation) s'exécutent en file d'attente sur la station.
