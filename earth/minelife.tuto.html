Règles WoTx2
Règle A — 3 Kind 7 (réactions endorsement)
Un joueur obtient le niveau X1 d'un skill quand 3 personnes différentes réagissent (+) à son annonce Kind 30501 avec un Kind 7 ciblant le bon permit. Chaque endorseur doit lui-même détenir le skill à un niveau ≥ X1.
Règle B — Kind 30502 (certification directe)
Un détenteur X1 peut certifier directement un autre joueur via un événement Kind 30502 (attestation signée). Une seule certification suffit pour le niveau X1 (mais n'octroie pas le level suivant sans validation supplémentaire).
Oracle — Kind 30503 (validation capitaine)
Le capitaine de la station signe un Kind 30503 basé sur la preuve Kind 30502. Ce Kind 30503 est le titre officiel inscrit dans la constellation. Les skills composites (ex: devops = linux+bash+docker) nécessitent un Kind 30503 additionnel publié par l'Oracle UPLANETNAME_G1.
