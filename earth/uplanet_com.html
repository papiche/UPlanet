Comment fonctionne la protection SSSS (Shamir) ?

Votre secret SALT + PEPPER est protégé par le Shamir's Secret Sharing Scheme :

  1. Le secret est divisé en 3 fragments
  2. 2 fragments sur 3 sont nécessaires pour reconstruire le secret
  3. Chaque fragment est chiffré asymétriquement pour un destinataire différent
👤
Fragment 1
Chiffré pour VOUS
(votre G1PUB)
⚓

(garant local)
🌍

(garant global)

Résultat : Même si vous perdez votre QR code, vous pouvez récupérer votre identité avec l'aide du Captain OU d'UPlanet. Aucun acteur seul ne peut accéder à votre secret.

Comment fonctionne le stockage IPFS (uDRIVE) ?

IPFS (InterPlanetary File System) est un système de fichiers distribué. Votre uDRIVE est un espace de 128 Go qui vous appartient :

  • Adressage par contenu : Chaque fichier a une empreinte unique (CID)
  • Pas de serveur central : Vos données sont répliquées sur l'essaim
  • IPNS : Une clé publique pointe vers votre espace (mise à jour dynamique)
  • Privé : Vos données ne sont pas scannées pour la pub

Synchronisation : Les stations Astroport forment un essaim privé qui réplique vos données automatiquement.
